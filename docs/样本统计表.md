# 样本统计表

## 目标

- 新增独立脚本 [sample_table.py](/Users/zhangqijin/PycharmProjects/hangbo/sample_table.py)
- 直接读取原始问卷目录，例如 `datas/Q1`
- 按现有客户类型映射规则统计每个样本类型的执行样本数
- 输出一个单独的 Excel 文件，不和满意度汇总表放在同一个工作簿

## 输入口径

- 映射规则继续复用 [survey_customer_category_rules.py](/Users/zhangqijin/PycharmProjects/hangbo/survey_customer_category_rules.py)
- 实际样本数来自原始问卷 `问卷数据` sheet
- 样本分组基于原始数据中的 `年份` + `月份` 两列组合
- 如果缺少 `年份` 列或单元格为空，默认按 `2026` 年处理
- 如果没有显式传入月份分组参数，会自动按命中的 `年份` + `月份` 组合生成分组列
- 自动生成的分组列按列名字面顺序排序
- 自动生成的分组列名会做展示格式化，例如 `01-02` 输出为 `1-2月`，`03` 输出为 `3月`

## 默认配置

默认配置文件是 [sample_table.default.toml](/Users/zhangqijin/PycharmProjects/hangbo/sample_table.default.toml)。

这里记录了：

- 表头标题
- 输出 sheet 名
- 默认输出文件名
- 每一行的展示顺序
- 每一行的目标样本量
- 每一行对应的映射规则名
- 特殊行的手工覆盖值

其中：

- 如果某一行配置了 `rule_name` 但没写 `display_name`，会自动取 [survey_customer_category_rules.py](/Users/zhangqijin/PycharmProjects/hangbo/survey_customer_category_rules.py) 里的 `customer_category`
- `customer_mapping.xlsx` 仍然是业务原始依据；Python 规则文件负责把这份依据转成可执行规则

后续如果样本量目标调整，优先直接改这个 TOML 文件。

## 特殊规则

- `会展流失主办客户`：实际样本数固定记 `0`
- `酒店散客`：实际样本数复用 `散客` 的映射结果
- `二、酒店暗访（次）`：实际样本数固定记 `1`
- `五、专项调研`：当前没有映射到原始问卷，默认按 `0` 输出
- 酒店客户 5 行的样本类型名称与满意度汇总表保持一致，默认顺序为 `酒店散客`、`酒店住宿团队`、`酒店参会客户`、`酒店会议主（承）办`、`酒店餐饮客户`

## 表格结构

输出列为：

- `客户大类`
- `样本类型`
- `样本量`
- `样本进度百分比`
- `总执行样本量`
- `月份分组列（动态）`

其中：

- 数据行的 `样本进度百分比` 使用公式 `总执行样本量 / 样本量`
- `总执行样本量` 保留总样本数口径，不受是否配置月份分组影响
- `月份分组列` 会按实际分组数量动态展开，例如 `1-2月`、`3月`
- 多行客户大类后面会追加 `小计`
- 最后一行会生成 `合计`
- `小计` 和 `合计` 的样本量、总执行样本量、样本进度百分比、月份分组列都尽量使用 Excel 公式

示例表头可能是：

- `客户大类 | 样本类型 | 样本量 | 样本进度百分比 | 总执行样本量 | 1-2月 | 3月`

## 运行方式

```bash
uv run python sample_table.py \
  --input-dir datas/Q1 \
  --output-dir 汇总结果/Q1
```

如果要换配置：

```bash
uv run python sample_table.py \
  --input-dir datas/Q1 \
  --output-dir 汇总结果/Q1 \
  --config sample_table.default.toml
```

如果要手动指定月份分组列名，可以重复传入 `--month-group`：

```bash
uv run python sample_table.py \
  --input-dir datas/Q1 \
  --output-dir 汇总结果/Q1 \
  --month-group '1-2月=01-02' \
  --month-group '3月=03'
```

分组参数说明：

- 格式一：`列名=月份`
- 格式二：`列名=年份:月份`
- 未写年份时，默认使用 `--default-year`，默认值为 `2026`

例如：

- `--month-group '1-2月=01-02'`
- `--month-group '3月=03'`
- `--month-group '2025年12月=2025:12'`

如果只想改默认年份，可以显式传入：

```bash
uv run python sample_table.py \
  --input-dir datas/Q1 \
  --output-dir 汇总结果/Q1 \
  --default-year 2025
```
