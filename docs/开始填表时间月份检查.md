# 开始填表时间月份检查

用于遍历指定目录下的所有 `xlsx` 文件，读取 `问卷数据` sheet 中的 `开始填表时间` 字段，判断：

- 所有有效时间值是否都属于同一个月
- 检测到了哪些月份
- 每个文件分别属于哪个月
- 哪些文件缺少 `问卷数据` sheet、缺少字段，或字段为空

脚本入口：

- [check_start_time_month.py](/Users/zhangqijin/PycharmProjects/hangbo/check_start_time_month.py)

## 用法

```bash
uv run python check_start_time_month.py --input-dir '1-2月原始数据'
```

递归扫描子目录：

```bash
uv run python check_start_time_month.py --input-dir '1-2月原始数据' --recursive
```

默认读取：

- sheet：`问卷数据`
- 字段：`开始填表时间`

如需覆盖，也支持显式指定：

```bash
uv run python check_start_time_month.py \
  --input-dir '1-2月原始数据' \
  --sheet-name '问卷数据' \
  --field-name '开始填表时间'
```

## 输出说明

脚本会输出两部分内容：

- 整体结论：是否都属于同一个月、检测到的月份、是否每个文件都能确认且只属于同一个月
- 文件明细：逐个文件列出月份和计数

常见结果说明：

- `2026-02`：该文件所有有效时间值都在 2026 年 2 月
- `2026-01, 2026-02（跨月）`：该文件内部同时出现了 1 月和 2 月
- `字段缺失`：`问卷数据` sheet 中没有 `开始填表时间` 列
- `无有效时间值`：字段存在，但这一列全为空或都无法解析
- `缺少 问卷数据 sheet`：文件里没有目标 sheet

## 适用约束

- 当前按你的数据约定，只检查 `问卷数据` sheet
- 会自动跳过 `~$` 和 `._` 这类 Excel 临时文件
