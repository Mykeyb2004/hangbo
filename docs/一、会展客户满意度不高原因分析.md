# 一、会展客户满意度不高原因分析

## 1. 分析说明

- 分析对象：`一、会展客户`
- 原始数据来源：`datas/1-2月`、`datas/3月`、`datas/Q1`
- 涵盖客群：`展览活动主（承）办`、`参展商`、`专业观众`、`会展服务商`、`会议活动主（承）办`、`参会客户`
- 口径说明：本分析基于 raw data 重新按现有问卷统计逻辑计算，重点看样本量、客群结构、低分维度、活动分布及开放题反馈
- 分析脚本：`test/analyze_exhibition_customer_satisfaction.py`

## 2. 结论摘要

### 2.1 总体判断

`一、会展客户`满意度不高，并不是因为核心会展服务能力整体失守，而是因为：

1. **大样本客群中的低分项集中暴露**，尤其是`专业观众`、`参会客户`、`参展商`
2. **拖累维度集中在餐饮、停车、标识导视、展厅/场地使用、休息空间等配套体验**
3. **3月高客流活动增加后，专业观众样本显著放大，导致“观众端体验短板”被充分显现**

### 2.2 Q1 样本加权视角下的总体表现

按样本量加权，Q1 `一、会展客户`整体满意度约为 **9.55**。各维度中：

- `产品服务`：**9.96**，表现最好
- `配套服务`：**9.71**，总体尚可
- `硬件设施`：**9.38**，偏弱
- `餐饮服务`：**9.17**，是最弱项

说明：会展客户的核心服务、人员服务、现场对接并不是主要问题，真正拉低整体感知的是到访端配套体验。

## 3. 哪些客群在拖累“一、会展客户”满意度

### 3.1 从样本结构看，三类客群决定了整体观感

Q1 样本量分布：

- `参会客户`：287 份，占 **48.8%**
- `专业观众`：150 份，占 **25.5%**
- `参展商`：118 份，占 **20.1%**

这三类客群合计占 Q1 `一、会展客户`样本的 **94.4%**，因此整体满意度高不高，主要取决于这三类客群。

### 3.2 从“拖累总体”的贡献看，专业观众最关键

按“与10分的差距 × 样本占比”估算，Q1 对整体满意度拖累最大的客群依次为：

1. `专业观众`：贡献约 **0.186** 分缺口
2. `参会客户`：贡献约 **0.137** 分缺口
3. `参展商`：贡献约 **0.094** 分缺口

其中，`专业观众`是最核心的拖累来源：样本量大，且满意度仅 **9.27**，明显低于参会客户（**9.72**）和参展商（**9.53**）。

## 4. 满意度不高，具体低在哪里

### 4.1 Q1 低分指标集中在“停车 + 餐饮 + 导视/展厅使用”

Q1 有效样本数较多、且得分偏低的指标包括：

- `参展商-园区停车方便`：**8.00**（17份有效样本）
- `专业观众-园区停车方便`：**8.08**（13份）
- `参会客户-园区停车方便`：**8.64**（39份）
- `专业观众-餐饮服务`：**8.88**（34份）
- `展览活动主（承）办-餐饮服务`：**8.90**（10份）
- `会议活动主（承）办-标识标牌清晰`：**8.92**（13份）
- `参展商-展厅使用情况`：**9.03**（118份）
- `专业观众-展厅使用情况`：**9.13**（150份）
- `专业观众-展会路线安排`：**9.11**（149份）
- `会展服务商-休息空间`：**8.88**（8份）

可以看到，问题并不在“销售经理、工作人员态度、沟通协调”这类核心服务项，而是集中在：

- 停车体验
- 餐饮体验
- 标识导视
- 展厅/场地使用便利性
- 休息空间

### 4.2 3月下降最明显的也是这些项

按 raw data 样本加权，`一、会展客户`在 3月 的表现相较 1-2月有明显回落：

- 样本加权总分：`9.75 → 9.38`
- 样本加权`产品服务`：`9.99 → 9.93`，仅小幅下降，仍维持高位
- 样本加权`硬件设施`：`9.51 → 9.29`
- 样本加权`配套服务`：`9.96 → 9.60`
- 样本加权`餐饮服务`：`9.57 → 8.70`，下降最明显

因此，从 raw data 看，3月会展客户满意度承压，本质上是**配套体验尤其是餐饮体验明显走弱**。

## 5. 哪类客群导致了 3 月表现变差

### 5.1 专业观众是 3 月最主要的下拉因素

1-2月 vs 3月 样本加权变化贡献中：

- `专业观众`：约 **-0.251** 分，为最主要下拉项
- `参会客户`：约 **-0.056** 分
- `参展商`：约 **-0.053** 分

专业观众之所以影响最大，有两个原因：

1. 其样本量从 **30** 份升至 **120** 份，占 3 月会展客户样本的 **35.8%**
2. 其满意度从 **9.81** 降至 **9.11**，下降 **0.70** 分

换言之，3月会展客户整体“不高”，本质上是**观众端样本放大后，观众端体验短板集中暴露**。

### 5.2 专业观众低分集中在哪里

3月专业观众低分项：

- `园区停车方便`：**8.08**
- `餐饮服务`：**8.29**
- `展厅使用情况`：**8.99**
- `展会路线安排`：**9.05**

对应到维度层面：

- `产品服务`：**9.96**（高）
- `硬件设施`：**9.19**（偏低）
- `配套服务`：**8.98**（偏低）
- `餐饮服务`：**8.29**（最低）

因此，专业观众不是对“服务人员”不满意，而是对**吃、走、找、歇**这四类到访体验更敏感。

### 5.3 参会客户和参展商的痛点也高度相似

- `参会客户`：停车 **8.57**、餐饮 **8.97**、会议标识偏弱
- `参展商`：停车 **8.00**、餐饮 **8.86**、展厅使用 **9.00**、标识偏弱

说明不同客群虽然身份不同，但低评价高度集中在同一类问题上：

- 停车与交通组织
- 标识导视
- 餐饮供给与价格/排队体验
- 展厅/场地使用便利性

## 6. 具体活动层面的信号

在 3 月、且单个活动样本量不低于 5 份的活动中，较低分活动包括：

- `参展商-消费者`：**8.97**（10份）
- `专业观众-沸点`：**9.10**（15份）
- `专业观众-美妆`：**9.31**（40份）
- `参会客户-人工智能`：**9.33**（21份）
- `参展商-母婴`：**9.39**（22份）

这说明 3 月低分并非完全分散，而是和部分高人流、高密度活动的现场体验有关。

## 7. 开放题反馈验证了上述判断

开放题反馈中，和低分项高度一致的主题包括：

### 7.1 专业观众反馈高频主题

- `动线交通`：5次
- `展厅场地`：5次
- `标识导视`：5次
- `休息空间`：3次
- `餐饮`：3次

代表性原始反馈包括：

- “从地铁口到展厅的路线太远”
- “到达展厅的路线有点远，标识摆放有点少”
- “可供休息的地方不太够”
- “吃饭不方便，排队时间过长，比较拥挤”

### 7.2 参会客户与参展商反馈主题

- `参会客户`：标识偏少、会议场地不好找、室内温度偏低
- `参展商`：地下车库上行标识偏少、餐饮价格偏贵、配套服务不够细

开放题反馈进一步说明：**会展客户满意度不高，主要是到访端体验摩擦较多，而不是核心服务能力下降。**

## 8. 综合判断

`一、会展客户`满意度不高，主要可以概括为以下几点：

1. **样本大头客群决定了整体观感**：参会客户、专业观众、参展商合计占比超过 94%
2. **专业观众是最主要拖累项**：样本占比高，且在餐饮、动线、导视、展厅使用上评价偏低
3. **多客群痛点高度同质化**：停车、标识、餐饮、场地使用、休息空间是共同短板
4. **核心会展服务并不是问题中心**：产品服务始终维持在高位，说明销售、对接、服务态度、业务能力整体稳定
5. **3月问题被放大，是高客流活动场景下的体验承压**：不是单点服务事故，而是运营承载下的配套体验问题

## 9. 汇报建议话术

建议对外或对领导汇报时表述为：

> 一、会展客户整体满意度不高，主要不是核心会展服务能力下降，而是高客流场景下观众端和到访端体验承压。尤其是专业观众、参会客户、参展商等大样本客群，在停车、标识导视、餐饮、展厅使用、休息空间等配套环节给出了相对更低评价，最终拉低了整体满意度。其中专业观众是最主要的影响来源。
