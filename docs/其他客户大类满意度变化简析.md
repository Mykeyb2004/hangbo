# 其他客户大类满意度变化简析

## 分析范围

- 二、餐饮客户
- 三、G20峰会体验馆
- 五、酒店客户

数据来源：

- `汇总结果/1-2月/1-2月客户类型满意度汇总表.xlsx`
- `汇总结果/3月/3月客户类型满意度汇总表.xlsx`
- `汇总结果/Q1/Q1客户类型样本统计表.xlsx`

分析脚本：`test/analyze_other_customer_groups.py`

## 结论摘要

### 1. 二、餐饮客户

- 是其他客户大类中下降最明显的一类
- 简单平均总分：`9.813 → 9.366`，下降 `0.447`
- 样本加权总分：`9.679 → 9.334`，下降 `0.345`
- 3月主要低分客户类型：`特色美食廊 9.10（23份）`、`婚宴 9.16（18份）`、`自助餐 9.46（8份）`
- 下降主要集中在：
  - `硬件设施`：`9.650 → 9.072`
  - `餐饮服务`：`9.977 → 9.654`
- 说明餐饮客户3月不仅有原有类型下滑，还叠加了新增低分类型进入统计范围，整体表现最弱

### 2. 三、G20峰会体验馆

- 1-2月无样本，3月开始出现样本，因此不能做严格环比
- 3月简单平均总分：`9.905`
- 3月样本加权总分：`9.829`
- 样本量仅 `10` 份，其中：
  - `游客`：`9.81`，9份
  - `旅行社工作人员`：`10.00`，1份
- 说明该大类当前满意度表现较好，但样本量很小，更适合继续观察，不宜做趋势判断

### 3. 五、酒店客户

- 整体仍维持较高水平，但较1-2月有一定回落
- 简单平均总分：`9.870 → 9.690`，下降 `0.180`
- 样本加权总分：`9.893 → 9.697`，下降 `0.196`
- `产品服务`基本稳定：`10.000 → 10.000`
- 下降主要集中在：
  - `硬件设施`：`9.737 → 9.423`
  - `配套服务`：`9.830 → 9.580`
  - `餐饮服务`：`9.877 → 9.610`
- 3月较弱客户类型为：`酒店餐饮客户 9.59（6份）`、`酒店散客 9.60（2份）`、`酒店参会客户 9.69（10份）`
- 酒店住宿团队仍保持高位：`9.88（5份）`

## 综合判断

- **下降最明显的是二、餐饮客户**，且无论按简单平均还是样本加权，结论一致
- **五、酒店客户整体仍偏高，但出现温和回落**，短板集中在硬件、配套和餐饮
- **三、G20峰会体验馆当前表现不错，但样本太少，趋势意义有限**

## 与会展客户的对比补充

若按3月样本加权总分看：

- 三、G20峰会体验馆：`9.829`（但样本仅10份）
- 五、酒店客户：`9.697`
- 一、会展客户：约 `9.384`
- 二、餐饮客户：`9.334`

说明3月除小样本的G20峰会体验馆外，酒店客户表现最好；餐饮客户表现最弱；会展客户和餐饮客户都承压，但餐饮客户回落更明显。
