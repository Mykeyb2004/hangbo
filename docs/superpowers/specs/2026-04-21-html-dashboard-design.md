# HTML Dashboard Design

**Goal:** 基于现有满意度分析结果，生成一个适合汇报演示的单页 HTML 仪表板，动态展示 1-2月 与 3月 在四个客户大类及客户分组上的满意度趋势变化。

**Architecture:** 使用单文件 HTML + ECharts 的方式构建前端可视化页面；数据由项目内测试脚本读取现有汇总表和样本统计表后输出为内嵌 JSON。页面采用单页三段式布局：顶部总览卡片、中部总体维度趋势、底部客户大类/客户分组联动图。交互以点击切换客户大类和自动动画过渡为主。

**Tech Stack:** HTML, CSS, JavaScript, ECharts, uv run Python

## 页面结构

1. 顶部概览区
   - 四个客户大类卡片：一、会展客户；二、餐饮客户；三、G20峰会体验馆；五、酒店客户
   - 每张卡片展示：1-2月加权均分、3月加权均分、变化值、3月样本量

2. 中部趋势区
   - 左图：官方汇总表底部五维度趋势（1-2月 vs 3月）
   - 右图：四个客户大类五维度样本加权对比图

3. 底部明细区
   - 客户大类切换按钮
   - 当前大类的客户分组横向哑铃图 / 分组对比图
   - 展示样本量和 1-2月→3月 变化，下降项高亮红色，上升项高亮绿色

## 交互与动画

- 页面加载时：卡片、图表逐步淡入
- 图表初始展示 1-2月，随后过渡到 3月
- 点击客户大类按钮时，底部图表平滑刷新
- Tooltip 展示：客户分组、样本量、维度值、变化值
