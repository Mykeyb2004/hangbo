# 跑Q1数据的命令



Q1 这套我实际用的是下面这 3 条主命令。

如果你要“直接覆盖旧结果”重跑：

```bash
uv run python survey_stats.py --config job_Q1.toml
uv run python summary_table.py --input-dir 输出结果/Q1 --output-dir 汇总结果/Q1 --output-name Q1客户类型满意度汇总表.xlsx
uv run python generate_ppt.py --config report_jobs.Q1.toml
```

对应含义是：

- 分项统计  
  配置文件：[job_Q1.toml](/Users/zhangqijin/PycharmProjects/hangbo/job_Q1.toml)  
  当前关键参数：
  - `input_dir = "datas/Q1"`
  - `output_dir = "输出结果/Q1"`
  - `output_format = "xlsx"`
  - `calculation_mode = "template"`

- 汇总统计  
  这次我是直接显式传参跑的：
  - `--input-dir 输出结果/Q1`
  - `--output-dir 汇总结果/Q1`
  - `--output-name Q1客户类型满意度汇总表.xlsx`

- PPT 生成  
  配置文件：[report_jobs.Q1.toml](/Users/zhangqijin/PycharmProjects/hangbo/report_jobs.Q1.toml)  
  当前关键参数：
  - `input_dir = "输出结果/Q1"`
  - `output_ppt = "输出结果/Q1满意度报告.pptx"`
  - `template_path = "templates/template.pptx"`
  - `section_mode = "auto"`
  - `chart_page.enabled = true`
  - `llm_notes.enabled = false`

如果你想先只校验、不落文件，可以用这两条：

```bash
uv run python survey_stats.py --config job_Q1.toml --dry-run
uv run python generate_ppt.py --config report_jobs.Q1.toml --dry-run
```
