# 问卷数据年月填充

用于遍历指定目录下的所有 `xlsx` 文件，只在 `问卷数据` sheet 上写入两个字段：

- `年份`
- `月份`

两个字段都按文本值写入。如果列已经存在，则直接覆盖原值；如果不存在，则在表头末尾追加新列。

脚本入口：
- [fill_year_month_columns.py](/Users/zhangqijin/PycharmProjects/hangbo/fill_year_month_columns.py)

## 适用场景

- 需要给一批原始问卷文件统一补上年份、月份字段
- 年份和月份由外部参数指定，不从文件内容自动推断
- 只希望修改 `问卷数据` sheet，保留工作簿中的其他 sheet

## 基本用法

```bash
uv run python fill_year_month_columns.py \
  --input-dir './datas/1-2月' \
  --year '2026' \
  --month '02'
```

递归处理子目录：

```bash
uv run python fill_year_month_columns.py \
  --input-dir './datas/1-2月' \
  --year '2026' \
  --month '02' \
  --recursive
```

自定义目标 sheet：

```bash
uv run python fill_year_month_columns.py \
  --input-dir './datas/1-2月' \
  --year '2026' \
  --month '02' \
  --sheet-name '问卷数据'
```

## 参数说明

- `--input-dir`：要处理的目录
- 示例里使用 `./datas/1-2月`，实际可替换为任何真实目录
- `--year`：写入 `年份` 列的文本值，例如 `2026`
- `--month`：写入 `月份` 列的文本值，例如 `02`、`2`、`2月`
- `--recursive`：是否递归处理子目录中的 `xlsx`
- `--sheet-name`：目标 sheet 名，默认 `问卷数据`

## 行为说明

- 仅处理扩展名为 `xlsx` 的文件
- 自动跳过 Excel 临时文件，如 `~$xxx.xlsx`
- 只修改目标 sheet，不重建整个工作簿
- 数据从第 2 行开始写入，第 1 行视为表头
- 即使原来单元格里是数字，也会被覆盖为文本值
- 如果文件缺少目标 sheet，会跳过该文件并在输出中提示
- 如果存在跳过文件，输出结尾会追加“跳过文件”列表，列出文件名和原因

## 输出示例

```text
目录: /path/to/input
扫描文件数: 2
更新成功: 1
跳过/失败: 1

文件明细：
- 展览.xlsx: 已更新 128 行
- 酒店.xlsx: 缺少 问卷数据 sheet
```
